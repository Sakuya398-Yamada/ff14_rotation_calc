# FF14 ローテーション計算ツール

FF14（ファイナルファンタジー14）のスキル回し（ローテーション）の **期待威力・PPS（Potency Per Second）を自動計算する Web ツール** です。

タイムライン上にスキルを並べるだけで、バフ・DoT・リキャストを加味した期待値がリアルタイムに算出されます。

---

## できること

- スキルをタイムラインに並べて **期待威力・PPS を自動計算**
- バフ・デバフの相乗を加味したダメージ期待値の算出
- DoT（継続ダメージ）・リキャストの可視化
- ステータス値（武器・主ステータス・クリティカル等）に応じた補正
- ボス離脱タイミングや PPS 集計範囲の指定

## 基本操作の流れ

1. **ジョブを選択する**
   - 画面左の「ジョブパレット」から対応ジョブを選ぶ
2. **スキルをタイムラインに配置する**
   - ジョブパレットに並んだスキルアイコンを、右側のタイムラインへドラッグ＆ドロップ
3. **タイムライン上で並び替える**
   - 配置済みのスキルをドラッグで左右に動かし、順序を調整
4. **期待威力・PPS を確認する**
   - 並べるたびに自動計算され、画面に反映されます

## 画面の説明

### ジョブパレット（画面左）

- ジョブ切替タブ
- そのジョブで使える攻撃スキル一覧（各スキルに威力を併記）
- **ステータス入力欄**: 武器ダメージ・主ステータス・クリティカル・ダイレクトヒット・意思力・スキル/スペルスピード等
- **レベル設定（α版）**: 初期状態は閉じています。Lv100 以外は威力値が保証外のためアルファ版扱いです

### タイムライン（画面右）

複数のレーンに分かれています。

| レーン | 内容 |
|--------|------|
| GCD | 通常攻撃・リキャスト中の主要スキル |
| oGCD | アビリティ（ウィービング枠） |
| バフ | 自己バフ・パーティバフの効果時間 |
| リソース | ジョブゲージ・スタック系リソース |
| DoT | 継続ダメージの効果時間 |
| リキャスト | 個別スキルのリキャスト状態 |

### 計算範囲の指定

- **PPS 範囲選択**: 威力総計・PPS を計算する時間範囲を指定できます
- **ボス離脱タイミング**: ボスが離脱（ターゲット不能）になる時刻を指定できます。離脱中は威力が発生しないため、計算から除外されます

## 制限事項

本ツールは「DPS ロールの単体ボスに対する火力ローテーション」の計算に特化しています。以下は **未対応** です。

- **味方対象のスキルは対象外**
  - 回復・軽減・味方バフ・シールド等は計算に含まれません（ヒーラー・タンクの支援系スキルは未対応）
- **方向指定・距離減衰は未考慮**
  - 側面／背面ボーナス、範囲スキルの距離減衰は反映されません
- **単体ボス想定**
  - 複数ヒット・複数対象の威力計算は、一部スキルのみ対応しています
- **レベルは Lv100 を前提**
  - レベル設定は α 版機能です。Lv100 未満では威力値が正確でない可能性があります
- **動作保証はしていません**
  - 公式の計算式・スキル仕様とズレがある可能性があります。最終的な判断はゲーム内実測と併用してください

## FAQ

### Q. 威力計算が公式や他ツールと微妙に違う

本ツールの計算式は独自実装です。バフの重複順序・小数処理・ヒット期待値モデルの違いで、数%単位のズレが出る場合があります。目的は **ローテーション同士の相対比較** であり、絶対値の厳密な一致は保証していません。

### Q. スキルのアイコンが欠けている / 表示されない

アイコン画像はジョブ・スキル追加時に個別に登録しています。未登録のものや URL 切れで欠ける場合があります。Issue 起票いただければ差し替えます。

### Q. 使いたいジョブが未対応

現状 WHM / DRG / BRD / PCT / BLM の 5 ジョブのみ対応しています。他ジョブは順次追加予定です。要望は Issue へどうぞ。

### Q. データは保存される？

ブラウザ内で動作しているため、タイムラインはリロードすると消えます。恒久的な共有・保存機能は未実装です。
